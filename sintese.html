GIT

É um software livre de versionamento que surgiu o desenvolvimento do Kernel do Linux, foi criado e desenvolvimento por Linus Torvalds em 2005.

Cada diretório de trabalho do Git é um repositório com um histórico completo e habilidade total acompanhamento das revisões, não depende de acesso a uma rede ou a um servidor central.

Comandos GIT


commit= torna permanente um conjunto de alterações em um repositório.
push= evia alterações para o servidor no repositório remoto.
fetch= verifica o seu repositório local está em sicronia com o remoto.
pull= sincroniza o repositório local com o remoto atualizando as versões.
