Tecnologias

HTML é o conteúdo, texto, tabela, imagem e vídeo

CSS é a estilização do conteúdo cores, fontes, tamanho, sombras e posicionamento.

JS é a interatividade menus interativos, validação, popups e animação.

TAGS

As tags definem ao navegador a marcação de conteúdo. As tags tem funcionalidades específicas. Sem o uso de tags o navegador exibe um texto plano.
Além disso as tags podem possuir atributos, que são parametros que definem algum tipo de informação extra ao navegador.

Marcação de títulos

As tags de heading definem o título e possui seis níveis de importância h1, h2, h3, h4, h5 e h6 (todos os H devem conter <>)

Marcação de parágrafos

As tags de parágrafo definem a apresentação dos parágrafos na sintaxe: <'p> Este é um parágrafo <'/'p>

<4>Marcação de quebra de linha

As tags de quebra de linha produzem uma quebra de linha em um texto. É útil para escrever poemas ou um endereço, onde a divisão de linha é significante. Sintaxe: Esta é uma quebra de linha <'b'r> .

Marcação para formatação de texto

As tags para formatação de texto definem destaque, tamanho de fonte, cor dentre outros.
